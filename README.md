# arbor-kids-savings

Static Vercel deploy of the BrandNewDay Kids Savings dashboard.

- 4 kids x 3 accounts (Papa & Mama / Opa & Oma / Tanya) = 12 BNDA accounts
- Balance history with multi-date editor
- Per-account BND projected-at-18 estimates
- Push-to-CashFlow feed via localStorage (`nijhuis_inv_cf_feeds`)

Source: `260223_Kids_Savings.html`, refreshed with 06-05-2026 BND PDFs.

Tile registered on the Nijhuis stack launcher (family-dashboard).
